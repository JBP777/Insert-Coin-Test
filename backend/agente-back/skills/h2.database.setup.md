# Skill: H2 Database Setup

## Objetivo
Configurar base de datos embebida H2 para desarrollo.

---

## Reglas
- Usar en fase de desarrollo
- No requiere Docker ni instalación externa
- Puede ser en memoria o en fichero

---

## Configuración básica

spring.datasource.url=jdbc:h2:file:./data/gamesdb
spring.datasource.driverClassName=org.h2.Driver
spring.jpa.database-platform=org.hibernate.dialect.H2Dialect
spring.h2.console.enabled=true

---

## Acceso consola
http://localhost:8080/h2-console

---

## Buenas prácticas
- Usar file mode para persistencia
- Usar mem mode para tests rápidos
