# Concert Management API Skill

## Objetivo
Gestionar conciertos mediante API REST.

## Endpoints mínimos
GET /api/concerts
GET /api/concerts/{id}
POST /api/concerts

## Campos comunes
- title
- artist
- venue
- date
- availableTickets

## Buenas prácticas
- Validar datos
- Respuestas JSON limpias
- Separar DTOs

## Requisitos que cubre
- Listar conciertos
- Ver detalle
- Crear conciertos
