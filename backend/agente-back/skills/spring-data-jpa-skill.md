# Spring Data JPA Skill

## Objetivo
Gestionar persistencia relacional usando Spring Data JPA e Hibernate.

## Responsabilidades
- Crear entidades JPA
- Gestionar repositorios
- Persistir datos en H2
- Implementar relaciones entre tablas

## Tecnologías
- Spring Data JPA
- Hibernate
- Jakarta Persistence

## Conceptos clave
- @Entity
- @Table
- @Id
- @GeneratedValue
- JpaRepository
- findAll()
- findById()
- save()
- deleteById()

## Buenas prácticas
- Mantener entidades simples
- Evitar lógica de negocio en entidades
- Usar repositories para acceso a datos
- Nombrar tablas correctamente

## Requisitos que cubre
- Persistencia relacional
- Operaciones CRUD
- Uso de base de datos relacional
