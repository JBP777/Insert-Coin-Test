# DTO Mapping Skill

## Objetivo
Separar entidades JPA de la API REST.

## Funcionalidades
- DTOs request/response
- Mapeo manual
- Validaciones en DTOs

## Buenas prácticas
- No exponer entidades directamente
- Mantener payloads simples
- Evitar relaciones complejas en respuestas

## Ejemplos
- CreateConcertRequest
- AttendeeResponse

## Requisitos que cubre
- API limpia
- Validaciones organizadas
