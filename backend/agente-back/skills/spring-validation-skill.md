# Spring Validation Skill

## Objetivo
Validar datos de entrada usando Bean Validation.

## Tecnologías
- Jakarta Validation
- Hibernate Validator

## Anotaciones clave
- @Valid
- @NotBlank
- @NotNull
- @Positive
- @Email
- @Size

## Casos comunes
- Nombre obligatorio
- Email válido
- Cantidad de entradas mayor que 0
- concertId obligatorio

## Buenas prácticas
- Validar siempre en DTOs
- Mostrar mensajes claros
- Evitar validaciones manuales repetidas

## Requisitos que cubre
- Validaciones correctas
- Calidad de datos
