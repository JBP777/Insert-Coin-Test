# Exception Handling Skill

## Objetivo
Gestionar errores backend correctamente.

## Herramientas
- @ControllerAdvice
- @ExceptionHandler

## Errores comunes
- ResourceNotFoundException
- ValidationException
- IllegalArgumentException

## Respuestas esperadas
404 NOT FOUND
400 BAD REQUEST

## Buenas prácticas
- Mensajes claros
- Respuestas consistentes
- No mostrar stacktrace al cliente

## Requisitos que cubre
- API robusta
- Manejo correcto de errores
