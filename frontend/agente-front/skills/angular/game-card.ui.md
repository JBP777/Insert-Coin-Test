# Skill: Game Card UI

## Objetivo
Crear un componente reutilizable para mostrar videojuegos en formato tarjeta.

## Entrada
- Game: { title, image, rating, platform }

## Tareas
- Crear componente standalone Angular
- Diseñar card responsive
- Mostrar imagen, título, rating y plataforma

## Reglas
- No incluir lógica de negocio
- No acceder a APIs directamente
- SSR-safe
